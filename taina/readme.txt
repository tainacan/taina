A beautiful block theme for WordPress digital archives with support for Tainacan.

== Description ==

An open-source, modern full-site-editing [block theme](https://developer.wordpress.org/themes/block-themes/) for your digital expositions that works with [Tainacan](https://tainacan.org/en).

Tainá is made for museums, libraries and galleries for sharing their stories with the World.

Help improve the project in [our GitHub repo](https://github.com/tainacan/taina/).

== Frequently Asked Questions ==

= Where do I report security bugs found in this theme? =

Please report security bugs found in the source code of the Tainá theme through the [Patchstack Vulnerability Disclosure Program](https://patchstack.com/database/vdp/9e5fc226-f254-4a93-83a4-fb9e4ae1eb25). The Patchstack team will assist you with verification, CVE assignment, and notify the developers of this theme.
