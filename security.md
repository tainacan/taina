# Where do I report security bugs found in this theme? 

Please report security bugs found in the source code of the undefined theme through the [Patchstack Vulnerability Disclosure  Program](https://patchstack.com/database/vdp/9e5fc226-f254-4a93-83a4-fb9e4ae1eb25). The Patchstack team will assist you with verification, CVE assignment, and notify the developers of this theme.
